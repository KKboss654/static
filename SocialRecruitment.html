社会招聘 校园招聘

  • 1、.NET（初级/中级/高级）

    岗位职责：

    参与.NET项目相应的需求讨论、项目设计、代码编写、文档编写、单元测试等工作。

    岗位要求：

    （1） 大专（含）以上学历，计算机相关专业，1年以上相关项目经验；

    （2） 熟悉WEB项目开发，掌握Ajax，HTML，CSS，XML，JavaScript，了解jQuery；

    （3） 熟悉Visual Studio开发环境，掌握ASP.NET/C#,或VB.NET；

    （4） 熟悉MS SQL Server开发或设计，熟悉存储过程、触发器。

  • 2.SharePoint开发工程师

    岗位职责：

    负责SharePoint核心项目的开发和维护，如企业项目管理系统。

    岗位要求：

    （1） 大专（含）以上学历，计算机相关专业，2年以上ASP.NET项目开发经验；

    （2） 熟悉WEB项目开发，掌握Ajax，HTML，CSS，XML，JavaScript，了解jQuery；

    （3） 熟悉Visual Studio开发环境，掌握ASP.NET/C#,或VB.NET；

    （4） 熟悉MS SQL Server开发或设计，熟悉存储过程、触发器；

    （5）还应了解过SharePoint开发机制，乐于/善于学习新技术；

    （6） 工作责任心强，具备良好的团队合作精神，较强的沟通、协作能力。

  • 3.Java（初级/中级/高级）

    岗位职责：

    参与Java项目相应的需求讨论、项目设计、代码编写、文档编写、单元测试等工作。

    岗位要求：

    （1） 大专（含）以上学历，计算机相关专业，1年以上相关项目经验；

    （2） 熟悉WEB项目开发，掌握Ajax，HTML，CSS，XML，JavaScript，了解jQuery；

    （3） 掌握J2EE相关技术，如SERVLET/JSP/JMS/JTA；

    （4） 熟悉spring、struts、ORMAPPING框架（Hibernate或IBATIS）；

    （5） 熟悉ORACLE，熟悉PL/ SQL开发；

    （6） 熟悉主流的J2EE应用服务器，如Weblogic、Websphere等；

    （7） 工作责任心强，具备良好的团队合作精神，较强的沟通、协作能力。

  • 4.软件测试工程师（初级/中级/高级）

    岗位职责：

    （1） 搭建测试环境，编写测试计划；

    （2）执行产品的功能、性能及其他方面的测试；

    （3） 对测试发现的问题进行跟踪，编写测试报告。

    岗位要求：

    （1） 大专（含）以上学历，1年以上软件测试经验；

    （2） 熟悉软件测试方法和流程，了解SQL script, VBScript, Browser and Database的相关知识；

    （3） 具备一定的缺陷管理意识，严谨的工作习惯；

    （4） 较好的文档编写能力，熟悉测试用例、测试报告的编写；

    （5） 较强的逻辑分析能力和学习能力，具备较强的总结能力；

    （6） 有性能测试和安全测试的经验者优先，了解使用QTP, MS VSTS等测试工具者优先。
